# 🛒 ShopSmart - E-Commerce App

## Overview

ShopSmart is a basic e-commerce frontend built using **Next.js** and **TypeScript**. It allows users to browse a list of products, view their details via dynamic routing, and log in to access authenticated functionality.

## 🔧 Features

- Product listing via **SSR**
- Dynamic product detail pages
- Image optimization using `next/image`
- Login page with basic JWT-based auth
- Global CSS styling and responsive layout
- Type-safe interfaces using TypeScript
